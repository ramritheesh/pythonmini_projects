# 🐍 Python Mini Projects

A collection of Python mini projects developed to strengthen programming fundamentals, automation skills, computer vision, and web scraping. These projects demonstrate practical applications of Python using libraries such as OpenCV, MediaPipe, BeautifulSoup, and more.

---

## 📂 Projects

| Project | Description |
|---------|-------------|
| 🎲 Dice Rolling Game | Simulates rolling a dice using Python's `random` module. |
| 🔢 Number Guessing Game | Interactive game where the user guesses a randomly generated number. |
| ✊ Rock Paper Scissors | Classic Rock-Paper-Scissors game played against the computer. |
| 🧮 Calculator | Command-line calculator supporting basic arithmetic operations. |
| 😊 Face Recognition | Detects faces using OpenCV's Haar Cascade classifier. |
| ✋ Gesture Volume Control | Controls system volume using hand gestures with MediaPipe and OpenCV. |
| 📁 Automatic File Organizer | Automatically organizes files into folders based on their extensions. |
| 🌐 Book Web Scraper | Scrapes book title, price, and rating from an online bookstore and stores the data in a CSV file. |
| 📱 QR Code Generator | Generates QR codes from text or URLs. |

## 🚀 Getting Started

### Clone the repository

```bash
git clone https://github.com/ramritheesh/pythonmini_projects.git
```

### Navigate to the project

```bash
cd pythonmini_projects
```

### Create a virtual environment (Optional)

```bash
python -m venv .venv
```

### Activate the virtual environment

#### Windows

```bash
.venv\Scripts\activate
```

#### macOS/Linux

```bash
source .venv/bin/activate
```

### Install the required packages

```bash
pip install opencv-python mediapipe beautifulsoup4 requests qrcode numpy
```

---

## ▶️ Running a Project

Run any project using Python.

Example:

```bash
python calculator.py
```

or

```bash
python gesture_volume_control.py
```

or

```bash
python web_scrapper.py
```
